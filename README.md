## Overview

Affinity Designer is a Windows vector graphics application built for producing polished artwork, UI assets, illustrations, and design compositions with accuracy at the center. It is intended for designers who want a desktop workflow for fine-grained shape editing, responsive handling, and export-ready results.

Release 2.6.0 combines the essential vector design toolkit with supporting features for multi-artboard projects, pixel-aware editing, and reusable asset management. Whether you are creating brand systems, digital illustrations, or production graphics, this version offers a practical workspace that carries projects from early concepts through final export.

## Key Features

- Multi-artboard support for keeping multiple layouts inside a single document
- Advanced pen tool for precise path building and curve shaping
- Non-destructive effects that let you keep iterating without committing changes too early
- Pixel perfect mode for tighter alignment in raster-oriented work
- GPU acceleration for smoother responsiveness on supported hardware
- Export persona for preparing files in multiple formats and dimensions
- Color blindness simulator for checking accessible color selections
- Asset management tools for organizing reusable design components
